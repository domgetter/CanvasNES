Paint program for Nintendo Entertainment System

Features:
  *Primary color selection
  Secondary color selection
  *3 palettes plus background (1 palette reserved for UI)
  *20x24 tile canvas
  *Mouse support
  *Ruler
  clear canvas

Possible features
  dotted line from ruler to cursor
  save files (still get save states with powerpak powermappers)
  emphasis bit selection
  tile fill
  undo
  shapes (lines, retangles, circles, etc.)
  music/sound
  zoom feature on bottom right
  change palette without changing color of a pixel
  Nice UI for selecting new color
